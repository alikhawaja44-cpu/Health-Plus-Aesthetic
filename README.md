# Health+ Aesthetic Clinic Management System

A comprehensive clinic management system built with React.js for managing patients, appointments, prescriptions, billing, and inventory.

## Features

- 🔐 **Secure Authentication System**
- 👥 **Patient Management** with MR Number generation
- 📅 **Appointment Scheduling** with token system
- 💊 **Prescription Management** with PDF export
- 💰 **Billing & Invoicing** system
- 🏥 **Pharmacy Inventory** with stock tracking
- 👨‍⚕️ **Staff Management** with role-based access
- 📱 **Responsive Design** for all devices

## Technologies Used

- React.js 18.2.0
- Tailwind CSS
- Lucide React Icons
- jsPDF for document generation
- Local Storage for data persistence

## Setup

Simply open `index.html` in a modern web browser to run the application.

## GitHub Pages Deployment

This project is configured for GitHub Pages deployment. The main application file is `index.html` in the root directory.
